# Week Number Column

In the MultiViewCalendar, you can render a column which displays the number of the weeks within the current `month` view.

###### Example

```html
    <div id="multiViewCalendar"></div>
    <script>
        $("#multiViewCalendar").kendoMultiViewCalendar({
            weekNumber: true,
            messages: {
                weekColumnHeader: "Wk"
            }
        });
    </script>
```
